    SweetPanda推薦

    好文不分先後順序，每篇文章都一輩子受用不盡！感謝大大們無私地分享＜(＿ ＿)＞

    • 廖洧杰 : 專業就是連麻瓜都聽得懂的溝通術 - 將專業的東西能夠講給對方懂，並且能夠透過各種描述方式讓對方能夠理解，那才是專業。

    • Will保哥 : 承接大型專案的兩難 - 盡量讓他們不要把工作的焦點放在「專案」上，而是「學習」上。

    • 黑暗執行緒 : 「好程式」跟你想的不一樣! 初讀「重構」有感 - 程式執行效率很重要，但好理解容易維護也很重要。

    • returns : 台灣軟體業業界現況詳解、軟體工程師指南 - 英文不行就像不會卍解一樣，掌握英文才能解放軟體工程師的所有力量。
